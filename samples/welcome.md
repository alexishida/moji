# Moji

A clean, cross-platform **Markdown** viewer and editor.

## Features

- Open a `.md` file with a single click
- Toggle **edit** mode with live preview
- Export to HTML and PDF
- Light / dark theme
- Multilingual UI (pt-BR, en, es)
